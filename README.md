# OPTIMIZATION-MODEL


## Description

This project solves a business optimization problem using Linear Programming (PuLP). The goal is to maximize profit while satisfying resource constraints.

## Problem Statement

A company produces two products:

* Product A
* Product B

Each product contributes to profit and consumes limited resources (machine time and labor).

## Objective

Maximize total profit subject to constraints.

## Methodology

* Define decision variables
* Formulate objective function
* Apply constraints
* Solve using PuLP optimizer

## Technologies Used

* Python
* PuLP (Linear Programming)

## Installation

pip install -r requirements.txt

## Run

python optimization.py

## Output

* Optimal number of Product A and B
* Maximum achievable profit

## Deliverable

A working optimization model demonstrating business decision-making using linear programming.
